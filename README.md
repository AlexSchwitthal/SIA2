# Projet EPA, SIA2


Pour créer un fichier config.php à la racine du dossier sous le format suivant :

	    <?php

      define('DB_SERVER', 'localhost');	  // serveur MySql
      define('DB_DATABASE', 'EPA');	      // nom de la base de données
      define('DB_USER', 'root');		      // nom d'utilisateur
      define('DB_PWD', '');               // mot de passe

       ?>

en changeant localhost, EPA, root, et '' part les valeurs de connexion à la base de données.
